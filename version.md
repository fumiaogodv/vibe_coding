# Vibe Hub - 任务打卡模块版本记录

## v1.0（当前版本）

### 核心功能
- **任务管理**：添加/删除周期性任务，设置名称与周期天数，起始日自动设为当天
- **今日打卡**：基于公式 `(今天 - start_date) % 周期天数 == 0` 自动判定今日需执行的任务
- **灵活勾选**：非今日任务默认不勾选，但允许手动勾选（不自律一天做两天的事）
- **无惩罚重置**：勾选非今日任务时，自动将该任务起始日重置为今天，等价于"周期重新开始计算"
- **执行细节**：勾选任务后可展开输入框，填写具体执行内容（如"哑铃飞鸟4组"）
- **全局随笔**：底部文本区记录全天感受、三餐、总结
- **Markdown 导出**：一键生成格式化日记，模态框预览 + 一键复制到剪贴板
- **数据持久化**：JSON 文件本地存储（`modules/task_tracker/data.json`）
